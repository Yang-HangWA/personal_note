```
To access the index, you can unpack the tuple within the for loop statement like this:

cities = ['New York', 'Beijing', 'Cairo', 'Mumbai', 'Mexico']

for index, city in enumerate(cities):
    print(f"{index}: {city}")
Code language: Python (python)
Output:

0: New York
1: Beijing
2: Cairo
3: Mumbai
4: Mexico
```
